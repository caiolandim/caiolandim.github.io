Disciplinas Banco de Dados


Desenvolvimento de Sistemas

Essa disciplina tem como objetivo capacitar os alunos no desenvolvimento de sistemas web e desktop, dando versatilidade ao profissional de Banco de Dados.


Linguagem SQL

Disciplina com uma dos recursos básicos para o desenvolvimento da profissão, a linguagem de consulta ao banco: SQL.


Ambiente de Desenvolvimento

Através dessa matéria que os alunos terão as primeiras indicações sobre como publicar seus trabalhos na internet.


Análise Exploratória de Dados

Disciplina que busca criar a habilidade nos alunos de analisar dados, as informações que estes disponibilizam e tirar conclusões analíticas.
